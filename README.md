# The Temporal and Sectoral Effects of Federal Funds Rate Changes on U.S. Employment
### A Multi-Industry Analysis, 1990–2024

**Authors:** Tianhuiqi Chen · Jinmin (Lexie) Li  
**Course:** ECO 481  
**Last updated:** 2024-12-20

---

## Overview

This repository replicates the empirical analysis from our paper examining how changes in the Federal Funds Rate (FFR) transmit to U.S. employment **over time** and **across industries** from 1990–2024. The pipeline covers:

- Aggregate employment–FFR correlation analysis with 1–30 month lags  
- Historical trend visualization across five major tightening cycles  
- Industry-specific employment rate evolution and sensitivity  
- Income composition, volatility, and monetary-policy sensitivity by sector

## Data Sources

| Dataset | Source | How to obtain |
|---|---|---|
| Federal Funds Rate | FRED (St. Louis Fed) | Included as `data/raw/FEDFUNDS.dta` |
| CPS microdata | IPUMS CPS | Download from [cps.ipums.org](https://cps.ipums.org/). Save as `data/raw/cps_raw.dta`. Required variables: `year`, `month`, `ind1990`, `empstat`, `labforce`, `wtfinl`, `incwage`, `incint`, `incdivid`, `incrent`, `cpi99`. |

## How to Replicate

```stata
// 1. cd to the repo root in Stata
cd "/path/to/Macroeconomics-Analysis"

// 2. Place cps_raw.dta in data/raw/

// 3. Run the master script — everything else is automatic
do code/00_master.do
```

Total runtime: ~5–15 minutes depending on dataset size.

## Code Module Summary

| Script | Purpose |
|---|---|
| `00_master.do` | Sets portable path globals, auto-installs packages, calls modules 01–04 in order |
| `01_data_prep.do` | Converts FFR CSV→DTA, merges CPS microdata, codes industries, employment & LF variables |

| `03_industry_analysis.do` | Annual industry employment rates, combined & panel charts, lead/lag sensitivity table |
